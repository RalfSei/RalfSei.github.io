### 3. Verarbeitung von Daten beim Besuch der Website

Beim Aufruf dieser Website überträgt Ihr Browser automatisch Daten an den Server von GitHub Pages. Dazu gehören:

- IP-Adresse  
- Datum und Uhrzeit des Abrufs  
- Aufgerufene Seite  
- Browsertyp und Version  
- Betriebssystem  
- Referrer-URL  

Diese Daten werden von GitHub automatisch in **Server-Logfiles** gespeichert.

**Ich selbst werte diese Daten nicht aus und habe keinen direkten Zugriff auf die Logfiles.**

GitHub verarbeitet die Daten gemäß seiner eigenen Datenschutzrichtlinie:  
https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement
